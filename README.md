### RAG Pipeline — Under the Hood

```
Document Ingestion  →  Semantic Chunking  →  Sentence-Transformer Embeddings
        ↓
   ChromaDB (vector store)  →  Semantic Vector Search
        ↓
   Prompt Engineering  →  Ollama (local LLM inference, IPEX-LLM accelerated)
        ↓
   Conversational Retrieval  →  Streamlit Interface
```

Fully local, end to end — no external AI API calls in the core pipeline.
